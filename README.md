# Tic Tac Toe Game

A **two-player Tic Tac Toe game** built with **Vanilla JavaScript, HTML, and SCSS**, featuring **Sign Up / Sign In**, **player statistics**, and **match history persistence** using Local and Session Storage.

---

## Features

- **Two-player gameplay**: Players can play on the same device.
- **Sign Up & Sign In**: Create and log in to player accounts.
- **Player Profiles**: Choose an avatar (emoji) and set a username.
- **Match Tracking**: Tracks wins, losses, draws, and total matches.
- **Match History**: View previous matches grouped by date.
- **Session Persistence**: Current round and scores are saved in `sessionStorage`.
- **Local Storage**: Player data persists across sessions.
- **Interactive UI**: Responsive and colorful interface with live feedback.
- **Customizable Experience**: Players select shapes (X/O) and avatars.

---

## Tech Stack

- **HTML5**
- **SCSS / CSS**
- **JavaScript (ES6+)**
- **Local Storage & Session Storage**
